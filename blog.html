JavaScript Important Concepts

1. What are the differences between Local Storage and Session Storage ??

The differences between Local Storage and Session Storage are mentioned bellow

Local Storage

  1. Data in local storage does not expires automatically.
  2. The Storage Capacity in Local Storage is 5 / 10 MB.
  3. There is no transfer of data to the server.
  4. The client can only read local storage data.

Session Storage

  1. Data in session storage expires after the end of session.
  2. The Storage Capacity in Session Storage is 5 MB.
  3. There is no transfer of data to the server.
  4. The client can only read local storage data.

2. What are the differences between Global Scope and Block scope in JavaScript ??

The differences between Global Scope and Block scope in JavaScript are :

Global Scope

  1. The variables defined outside of any function or curly brackets { } are known as global variables and The scope the have is called global scope
  2. Variables that are declared in global scope can be accessed from anywhere in JavaScript Code
  3. If we declare variable with var from anywhere of the code it goes to global Scope.

Block Scope

  1. Generally, whenever we use curly brackets { } in code it creates a block and variables inside the block are in block scope.
  2. Variable that are declared inside a block scope can only be accessed inside that block
  3. ES6 provides the let and const keywords that allow us to declare variables in block scope.
